# College Management System

A **BCA Major Project** built using **Django**, **Python**, **OpenCV**, and **Docker**.

## Features

* Facial Recognition Attendance
* Drag & Drop Timetable Management
* Role-Based User Management
* Real-Time Chat
* Excel Import
* OTP Password Reset (Repalce email and password in settings.py)
* Student, Faculty, Coordinator, and Admin Modules

---

## Technologies Used

* Python
* Django
* OpenCV - facial-recognition
* Django Channels
* Redis
* Docker & Docker Compose

---

## Running the Project

### Build the Docker Image

```bash
docker compose build
```

### Start the Project

```bash
docker compose up
```

The application will be available at:

```text
http://localhost:8000
```

### Stop the Project

```bash
docker compose down
```
